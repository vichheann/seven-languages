# Seven-languages

Homework for "Seven languages in seven weeks" book. So this is the Io week ...

## Usage

If you use homebrew, you know what to do:

    brew install io

Otherwise, check the [project](http://iolanguage.com/)

And then simply launch script like this

    io day1.io
